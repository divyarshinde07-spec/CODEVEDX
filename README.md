# Student Performance Analysis using Python

## CODEVEDX Data Analysis Internship

### Project Overview

As part of my CODEVEDX Data Analysis Internship, I completed a Student Performance Analysis project using Python. The objective of this project was to analyze student performance data, identify trends, and generate meaningful insights through statistical analysis and data visualization.

### Objective

* Analyze student performance data.
* Calculate average scores in Mathematics, Reading, and Writing.
* Identify highest and lowest scoring students.
* Compare student performance based on gender.
* Generate visualizations to understand data trends.
* Extract meaningful insights from the dataset.

### Tools and Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* VS Code / Jupyter Notebook
* GitHub

### Dataset Information

The dataset contains student records with the following attributes:

* Gender
* Math Score
* Reading Score
* Writing Score

Total Records: 30 Students

### Analysis Performed

1. Data Loading and Exploration
2. Missing Value Analysis
3. Duplicate Record Removal
4. Statistical Summary Generation
5. Average Score Calculation
6. Highest and Lowest Scorer Identification
7. Gender-wise Performance Analysis
8. Correlation Analysis
9. Data Visualization

### Visualizations Generated

* Gender Distribution Chart
* Math Score Distribution
* Reading Score Distribution
* Writing Score Distribution
* Gender vs Math Score Boxplot
* Correlation Heatmap
* Top 10 Students by Total Score

### Key Findings

* Female students performed better in Reading and Writing.
* Reading and Writing scores showed a strong positive correlation.
* Most students scored between 60 and 90 marks.
* Top-performing students were successfully identified using total scores.
* Data visualization made it easier to understand student performance trends.
